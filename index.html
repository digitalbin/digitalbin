billigprilla.se

A website for comparing snus prices online

This was built with Gatsby and graphql on the frontend, backed by an Express REST API with MongoDB.

I also built a price scanner that I hosted on a raspberry pi, which updated each price a couple of times a day, which automatically regenerated the static pages if it detected any price change.

Note: This project isn’t maintained anymore so expect dead links and outdated information.
